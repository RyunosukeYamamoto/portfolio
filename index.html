Groomings(仮)

開発開始日

2021年8月24日~

説明

男性の身だしなみに関する悩みを他ユーザーと共有して解決したり、人気のある身だしなみに詳しいユーザーに相談したりできるアプリ。

主な機能

  • 身だしなみに関する悩みの投稿とそれに対するコメント
  • 人気ユーザーへの秘密の相談(他ユーザーからは見えない)
  • ポイント制による悪質ユーザーの排除(一定のポイント以上を持っていないとコメントや秘密の相談はできない)
  • 通知機能
  • ユーザーのポイント数 他のランキング表示
  • お気に入り機能

MensGrooming

開発開始日

説明

Clother

開発開始日

2021年2月17日~

説明

その日着る服を決めるのがめんどくさいという悩みを解決するために開発。手持ちの服を登録しておくと、現在の気温から自動でコーデを提案してくれる。

主な機能

  • ユーザーログイン機能
  • 現在の気温と天気の取得(openweathermapAPIを使用)
  • 服の登録(画像アップロードにはcarrierwaveを使用)※Heroku上で、画像をアップロードすると数分後消えてしまいます。(awsの有料サービスS3による解決を確認済み)
  • 服のカテゴリー分類(複数のカテゴリーに分類可)
  • カテゴリーの気温パターンへの分類(10~15°Cのときはこの組み合わせなどという感じ)(複数の気温パターンに分類化)
  • 服の提案(現在の気温と登録した気温パターンをもとに自動で服を提案)

使用技術

Ruby on Rails/HTML/CSS/JavaScript/JQuery/Bootstrap

EcoShow

開発開始日

2021年1月19日~

説明

身近なところからCO2排出量の削減に興味を持ってもらいたいという思いから作成。CO2排出量削減のためのアクションを自動車の走行距離に換算することで、より削減量がイメージしやすいようにした。

主な機能

  • ユーザログイン機能
  • エコアクションとそれによるCO2削減量の登録、編集、削除
  • ユーザのCO2削減量を自動車の走行距離に換算
  • 目標の登録とそれまでの到達度の表示
